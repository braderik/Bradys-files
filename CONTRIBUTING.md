# Contributing

Thank you for your interest in contributing to Bradys-files!

- Fork the repository and create a new branch for your work.
- Make your changes and include tests or documentation updates when relevant.
- If you wish to be acknowledged, propose an acknowledgments section in `README.md` as part of your pull request.
- Submit a pull request describing your changes.

By contributing, you agree that your contributions will be licensed under the MIT License.
